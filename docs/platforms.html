Platforms

PhysiCore ships 12 built-in platform dynamics functions plus 4 composite platforms. Each is accessed by name via PhysiCore.for_platform() or PLATFORM_DYNAMICS[name].

Core Platforms

Platform Parameters

balancing_bot

params = {
    "mass": 1.0,      # kg — total robot mass
    "friction": 0.15, # Nm·s/rad — rotational friction
    "inertia": 0.01,  # kg·m² — moment of inertia
}

quadrotor

params = {
    "mass": 1.5,          # kg
    "arm_length": 0.25,   # m — motor-to-center distance
    "k_thrust": 3.16e-6,  # N/(rad/s)² — thrust coefficient
    "k_torque": 7.94e-9,  # Nm/(rad/s)² — torque coefficient
    "Ixx": 0.0213,        # kg·m² — roll inertia
    "Iyy": 0.0213,        # kg·m² — pitch inertia
    "Izz": 0.04,          # kg·m² — yaw inertia
}

satellite

params = {
    "Ixx": 10.0,  # kg·m²
    "Iyy": 12.0,
    "Izz": 8.0,
    "max_torque": 0.1,  # Nm — reaction wheel saturation
}

Custom Dynamics

Any callable matching the signature f(state, action, params) → xdot can be used as a platform:

def my_dynamics(state, action, params):
    # state: np.ndarray shape (state_dim,)
    # action: np.ndarray shape (action_dim,)
    # params: dict of physical parameters
    # returns: xdot np.ndarray shape (state_dim,)
    ...

engine = PhysiCore(cfg, dynamics_fn=my_dynamics, platform_params=params)
