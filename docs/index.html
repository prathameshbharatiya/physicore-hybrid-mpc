What is PhysiCore?

PhysiCore is a production-grade Model Predictive Control (MPC) engine for real robots. It combines physics-based dynamics, residual neural networks, online system identification, and CEM optimization into a single cohesive runtime that works on any platform — from balancing bots to satellite attitude control.

30-Second Example

import numpy as np
from physicore import PhysiCore, PLATFORM_DYNAMICS

# Build engine for a balancing bot
engine = PhysiCore.for_platform("balancing_bot", {"mass": 1.0, "friction": 0.15, "inertia": 0.01})

# Simulate 100 MPC steps
x = np.array([0.1, 0.0, 0.0, 0.0])  # [angle, ang_vel, pos, vel]
for _ in range(100):
    step = engine.step(x, np.zeros(1))
    xdot = PLATFORM_DYNAMICS["balancing_bot"](x, step.action, {"mass": 1.0, "friction": 0.15, "inertia": 0.01})
    x = x + 0.02 * xdot

print(f"Final angle: {x[0]:.4f} rad")
