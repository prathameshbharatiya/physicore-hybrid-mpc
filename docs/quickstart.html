Quick Start

Get PhysiCore running in under 5 minutes.

Prerequisites

  • Python 3.9 or later
  • Node.js 18+ (for the React dashboard)
  • Git

Option A — One-Line Installer (Linux / macOS)

curl -fsSL https://raw.githubusercontent.com/prathameshbharatiya/physicore-hybrid-mpc/main/install.sh | bash

Option B — From Source

# 1. Clone the repository
git clone https://github.com/prathameshbharatiya/physicore-hybrid-mpc.git
cd physicore-hybrid-mpc

# 2. Run the installer (handles pip + verification)
python install.py

# 3. Install frontend dependencies
npm install

Option C — pip

pip install physicore[all]

Start the Server

# Using the CLI
physicore serve

# Or directly with uvicorn
uvicorn physicore.api.server:app --host 0.0.0.0 --port 8000

# Start the React dev server (separate terminal)
npm run dev
Open the dashboard at http://localhost:5173 and the API at http://localhost:8000/docs

Your First MPC Step

import numpy as np
from physicore import PhysiCore, PLATFORM_DYNAMICS

# Create engine for a balancing bot
engine = PhysiCore.for_platform(
    "balancing_bot",
    params={"mass": 1.0, "friction": 0.15, "inertia": 0.01}
)

# Initial state: [angle, angular_velocity, position, velocity]
x = np.array([0.1, 0.0, 0.0, 0.0])

# Run 200 MPC steps
for t in range(200):
    step = engine.step(x, np.zeros(engine.cfg.action_dim))
    xdot = PLATFORM_DYNAMICS["balancing_bot"](x, step.action, engine.platform_params)
    x = x + engine.cfg.dt * xdot
    if t % 50 == 0:
        print(f"t={t:3d}  angle={x[0]:+.4f} rad  residual={step.residual_norm:.5f}")

print("Done!")

Run an Example

# Run a built-in example
physicore run balancing_bot_sim
physicore run quadrotor_sim
physicore run fleet_two_robots
physicore run full_pipeline

# Or run directly
python examples/balancing_bot_sim.py
