Plugin SDK

PhysiCore plugins extend the engine at runtime — adding perception adapters, custom safety checks, analytics, or actuation overrides. Every plugin is scanned for security before installation.

Scaffold a Plugin

physicore plugins new my-plugin --author alice
# Creates: my-plugin/manifest.json, my-plugin/plugin.py, my-plugin/README.md

Plugin Interface

from physicore.sdk.plugin_loader import PhysicorePlugin, PluginMeta

class MyPlugin(PhysicorePlugin):

    @property
    def meta(self) -> PluginMeta:
        return PluginMeta(
            id="my-plugin",
            name="My Plugin",
            version="1.0.0",
            description="Does something useful",
            author="alice",
        )

    def on_load(self):
        """Called once when the plugin is activated."""
        pass

    def on_step(self, state, action, dt: float) -> dict:
        """Called every control step. Return a dict of extra telemetry."""
        return {"my_metric": 42.0}

    def on_unload(self):
        """Called when the plugin is deactivated."""
        pass

Security Scanning

Before a plugin is stored in the marketplace, scan_source() performs two passes:

Pass 1 — Regex patterns

Blocks any use of: socket, subprocess, ctypes, os.system, eval(, exec(, __import__, requests, urllib, aiohttp, httpx.

Pass 2 — AST walk

Walks every import and from … import statement, blocking the same module list at the AST level (catches aliased imports).

Exception: open('/plugins/...') is allowed for reading local plugin data files. 
from physicore.sdk.marketplace import scan_source

result = scan_source(source_code, filename="plugin.py")
print(result.safe)       # True / False
print(result.violations) # list of {line, pattern, snippet}

Submitting to the Marketplace

from physicore.sdk.marketplace import get_marketplace

mkt = get_marketplace()
entry_id = mkt.submit(
    manifest=my_manifest,
    source_files={"plugin.py": open("plugin.py").read()},
    author_id="alice",
    category="analytics",
    price=0.0,
)
print(f"Submitted: {entry_id}")

Installing a Plugin

# Via CLI
physicore plugins install gravity-comp

# Via Python
from physicore.sdk.marketplace import get_marketplace
mkt = get_marketplace()
install_path = mkt.install("gravity-comp", user_id="alice")

Loading a Plugin at Runtime

from physicore.sdk.plugin_loader import PluginLoader

loader = PluginLoader()
loader.load_from_path("/path/to/my-plugin/plugin.py")

# All loaded plugins receive on_step() calls
for plugin in loader.active_plugins():
    result = plugin.on_step(state, action, dt=0.02)
    print(result)
