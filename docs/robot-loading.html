Robot Loading

PhysiCore supports three ways to define a robot: built-in platform dynamics, URDF files, and custom dynamics functions.

Option 1 — Built-in Platform

The fastest way to get started. Choose from 12 pre-built platforms:

from physicore import PhysiCore

engine = PhysiCore.for_platform(
    "quadrotor",
    params={"mass": 1.5, "arm_length": 0.25, "k_thrust": 3.16e-6,
            "k_torque": 7.94e-9, "Ixx": 0.02, "Iyy": 0.02, "Izz": 0.04}
)

Option 2 — URDF File

from physicore.core.urdf_loader import load_robot, build_robot_model

# Parse the URDF — returns joint/link structure
model = load_robot("my_robot.urdf")
print(f"Robot: {model.robot_name}, {model.n_joints} joints")

for joint in model.joints:
    print(f"  {joint.name}: {joint.joint_type}  limits=[{joint.limit_lower:.2f}, {joint.limit_upper:.2f}]")

# Build a numerical model (FK, Jacobian, IK)
robot = build_robot_model("my_robot.urdf")

import numpy as np
q = np.zeros(robot.n_joints)
T = robot.forward_kinematics(q)   # 4×4 homogeneous transform
J = robot.jacobian(q)             # 6×n Jacobian

Option 3 — YAML Config

physicore robot new my_arm --platform manipulator_arm
# Generates my_arm.yaml
# my_arm.yaml
robot:
  name: my_arm
  platform: manipulator_arm
  urdf: path/to/arm.urdf   # optional

physics:
  mass: 2.0
  friction: 0.05
  inertia: 0.02

control:
  state_dim: 8
  action_dim: 4
  dt: 0.01
  horizon: 25

mpc:
  optimizer: cem
  samples: 512
  elites: 64
  iterations: 5

bridge:
  type: sim   # sim | serial | mavlink | ros2
  connection: null

plugins: []
physicore robot load my_arm.yaml

Option 4 — Custom Dynamics Function

import numpy as np
from physicore import PhysiCore, PhysiCoreConfig

def my_dynamics(state, action, params):
    """Returns xdot = f(x, u, params)."""
    x, v = state[0], state[1]
    u = action[0]
    m = params.get("mass", 1.0)
    xdot = np.array([v, (u - params.get("friction", 0.1) * v) / m])
    return xdot

cfg = PhysiCoreConfig(state_dim=2, action_dim=1, horizon=20, dt=0.02)
engine = PhysiCore(cfg, dynamics_fn=my_dynamics, platform_params={"mass": 1.0, "friction": 0.1})

JointInfo Fields

FieldTypeDescription
namestrJoint name from URDF
joint_typestrrevolute, prismatic, fixed, continuous
parentstrParent link name
childstrChild link name
origin_xyzarray[3]Translation offset
origin_rpyarray[3]Rotation offset (roll-pitch-yaw)
axisarray[3]Joint rotation/translation axis
limit_lowerfloatLower position limit (rad or m)
limit_upperfloatUpper position limit
limit_effortfloatMax torque/force (N·m or N)
limit_velocityfloatMax velocity (rad/s or m/s)
